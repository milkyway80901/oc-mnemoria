## 📥 Introduction

Welcome to "oc-mnemoria," a tool designed to help OpenCode agents store and access shared memory easily. This application uses the powerful mnemoria Rust engine to offer a seamless experience for users looking for advanced memory solutions.

## 💾 Features

- **Persistent Memory**: Store data even after you close the application.
- **Shared Access**: Allow multiple OpenCode agents to share memory spaces.
- **Easy Integration**: Work smoothly with OpenCode applications and plugins.
- **User-Friendly Interface**: Designed for quick understanding and easy navigation.

## 📋 System Requirements

Before downloading, ensure your system meets these requirements:

- **Operating System**: Windows 10 or later, macOS 10.15 or later, or a current Linux distribution.
- **RAM**: At least 4 GB required.
- **Disk Space**: Minimum of 100 MB available space.
